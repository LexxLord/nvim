netrw:
    %       - Создать новый файл
    -       - Предыдущая директория
    t       - Открыть файл в новом табе
    d       - Новая директория

gt          - Перемещение по табам вперёд
gT          - Перемещение по табам назад


Перемещение по строкам
Удобно с прямой нумерацией
gg          - 1-я строка
G           - последняя строка
4gg         - 4 строка

Удобно с относительной нумерацией
4j          - на 4 строки вниз
4k          - на 4 строки вверх
или номер и стрелки

zz          - Центрировать строку

Ctrl-U      - Пролистать вверх
Ctrl-D      - Пролистать вниз


Перемещение по строке
w           - на слово или символ разделитель вперёд
b           - на слово или символ разделитель назад
W           - на слово c
B           - на слово назад

f<символ>   - переместится к первому вхождению <символ> (f, f: ...f)
;           - следующее вхождение <символ>
,           - предыдущее вхождение <символ>


vib         - Выделить всё в : ()
viB         - ... : {}
vi'         - ... : ''
vi"         - ... : ""
vi(         - ... : ()
vi[         - ...: []

i <-> a     - внутри <-> включительно



:+p         - Вставить из системного буфера
:+y         - Скопировать в системный буфер



ALT-l                   - Клавиатура ENG/RUS



<leader>k               - Help
